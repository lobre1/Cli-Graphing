# Cli-Graphing
Graphing tool in the cli based on 2d arrays.
![Example](./screenshot_021.png)
## How to use
`./cligraph "(function)" (Width) (Height) (Scale Y) (Scale X)`
example:
`./cligraph "sin(x)" 200 50 10 0.1`
